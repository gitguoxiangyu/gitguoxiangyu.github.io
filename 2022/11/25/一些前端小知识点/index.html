属性选择器

  • CSS 属性选择器通过已经存在的属性名或属性值匹配元素

  • <style>
    .test[echo]{
    color: red;
    }
</style>
<body>
  <div class="test" echo>从此烟雨落京城，一人撑伞两人行</div>
  <div class="test">从此烟雨落京城，一人撑伞两人行</div>
</body>

  • Vue中中可以通过 scoped 给每个组件赋予单独的样式，这个是通过Vue给加了scoped组件中的每个标签都加了一个特定不重复的data属性，然后将该组件的style都加上这个data属性的属性选择器实现的
